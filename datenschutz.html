Diese Webseite wird über GitHub Pages gehostet. Beim Besuch dieser Webseite werden automatisch Informationen durch den Hosting-Anbieter (GitHub, Inc.) erhoben und gespeichert. Die Seite selbst sammelt an sich keine weiteren Daten.

API Playground & Externe Aufrufe

Wichtiger Hinweis: Beim Benutzen des API Playgrounds oder beim Aufrufen einer API-Adresse über dieses Tool wird der Request direkt vom Gerät des Nutzers (Client-Side) ausgeführt.

Der aufgerufene API-Dienst bzw. die Ziel-URL kann dadurch Daten wie Ihre IP-Adresse, Browser-Informationen sowie die übermittelten Header und Body-Inhalte einsehen.

Diese Übermittlung erfolgt direkt zwischen Ihrem Browser und dem Ziel-Server. api-sandbox.de fungiert hierbei lediglich als Interface und speichert diese Anfragen nicht zwischen.
